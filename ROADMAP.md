# 🗺️ PROJECT ROADMAP - Neoma Kwitansi

Visi pengembangan sistem kwitansi **Paguyuban Dharma Putra Mahesa**.

---

## ✅ Phase 1: Foundation & Stability (COMPLETED)
- [x] Migrasi ke Next.js App Router.
- [x] Integrasi Vercel Postgres (Neon Database).
- [x] Implementasi Server-side API untuk keamanan data.
- [x] Pembuatan UI Glassmorphism Premium.
- [x] Sistem Cetak PDF Presisi (210mm x 105mm).
- [x] Fitur Single & Bulk Entry.
- [x] Dashboard Rekapitulasi & Statistik.

## 🚧 Phase 2: Feature Enhancement (IN PROGRESS)
- [ ] **Export Advanced**: Export rekap bulanan ke format Excel/PDF Pro.
- [ ] **Multi-User Admin**: Role-based access control (Ketua, Bendahara, Sekretaris).
- [ ] **Automated Backup**: Backup database mingguan otomatis ke Cloud Storage.
- [ ] **Email Notification**: Kirim bukti kwitansi otomatis via Email.

## 🚀 Phase 3: Scaling & Mobile Integration
- [ ] **Progressive Web App (PWA)**: Akses offline dan instalasi di homescreen HP.
- [ ] **OCR Scanning**: Input data otomatis lewat foto nota fisik.
- [ ] **Integrasi Payment Gateway**: Terima donasi digital (QRIS) yang langsung menerbitkan kwitansi.

## 🎨 Design Vision
- Mempertahankan estetika **Glassmorphism** dengan fokus pada aksesibilitas.
- Memastikan performa **Turbopack** tetap optimal untuk *build* cepat.
- Mengutamakan **Mobile-First Experience** tanpa mengorbankan kualitas cetak desktop.
